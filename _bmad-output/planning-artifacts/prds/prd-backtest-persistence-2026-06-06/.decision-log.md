# Decision Log - SQLite Backtest Persistence PRD

This log tracks all architectural decisions, assumptions, and revisions made during the design and lifecycle of the SQLite Backtest Results Persistence PRD.

---

## [2026-06-06] - Initial Setup & Scope Alignment

### DEC-001: Language Standard Execution
- **Decision:** The PRD, decision logs, and all downstream artifacts will be written in **English** instead of French. All user interaction will also continue in English.
- **Rationale:** Ensures continuity with the Java/JavaScript codebase and adheres to the developer rules.

### DEC-002: GUI Integration in MVP
- **Decision:** The Electron desktop frontend (Vue 3 + Vite) must display, search, and filter the backtest results. It is included in the MVP scope, not deferred.
- **Rationale:** Direct user request to have the frontend in the MVP to easily consult and filter historical runs.

### DEC-003: Strategy Parameter Persistence
- **Decision:** Strategy input parameters will be persisted in the database.
- **Rationale:** Allows users to know exactly what parameters (e.g., SMA periods) produced the profitability metrics. We will store parameters as a flexible JSON string (`parameters` TEXT) in SQLite to support different strategy structures.

---

## [2026-06-06] - PRD Finalization

### DEC-004: Final Sign-off
- **Decision:** Mark the SQLite Backtest Results Persistence PRD status as final.
- **Rationale:** Gaps and scope adjustments (English translation, Vue 3 GUI included in MVP, parameters persisted) have been successfully resolved and updated in the PRD.

### DEC-005: Reusing the Existing SQLite Database File
- **Decision:** Do not create a new database file for backtests. Store the new `backtest_runs` table in the existing `events.db` SQLite database file (resolved by `RuntimeDataPaths.defaultEventStorePath()`) and ensure WAL (Write-Ahead Logging) is enabled.
- **Rationale:** Minimizes file clutter, leverages the existing runtime persistence paths, and guarantees concurrent access capabilities via SQLite WAL mode.
