# Incident Severity Matrix

| Severity | Description | Escalation Path |
|----------|-------------|-----------------|
| P0 | Critical outage affecting all strategies (e.g., platform-wide DB corruption) | PagerDuty → On‑call SRE → Architecture lead |
| P1 | Major impact on a single broker or large subset of strategies (e.g., broker disconnect) | SRE → Team lead |
| P2 | Limited impact affecting one strategy or non‑critical component (e.g., rate‑limit) | SRE on‑call |
| P3 | Minor inconvenience, no functional impact (e.g., UI typo) | Ticket in backlog |

*Table can be expanded with contact details as needed.*
